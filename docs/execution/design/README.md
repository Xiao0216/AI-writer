## 使用规则

- 没有完成 Quest 0 的设计交付，不允许进入任何开发 Quest。
- 设计稿必须覆盖产品主链路，而不是只画首页。
- 每张关键页面都必须能回答：
  - 这一页的核心任务是什么
  - 用户第一眼看什么
  - 用户下一步做什么

## 当前最小完成标准

学生至少需要完成：

- Figma 文件结构
- 低保真线框
- 5 张高保真关键页面
- 交互状态说明
- 设计交付日志
