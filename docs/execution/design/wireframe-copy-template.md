# 低保真线框可复制文本模板

> 用法：
> - 学生可以把下面的结构直接复制到 Figma 线框稿中
> - 不要求视觉精致，但必须保证页面结构和文案锚点存在

---

## 1. 登录页线框模板

### 页面标题

`进入文枢AI`

### 页面说明

`不是帮你写一段，而是帮你把整本书控住。`

### 模块顺序

1. 品牌区
   - 文枢AI
   - 长篇小说创作控制站
2. 微信登录区
   - 按钮：`使用微信扫码登录`
3. 开发登录区
   - 区块标题：`开发环境快捷登录`
   - 输入项：`openid` / `笔名` / `常写品类`
   - 按钮：`进入测试环境`
4. 状态提示区
   - 成功提示占位
   - 失败提示占位

---

## 2. 项目控制台线框模板

### 页面标题

`创作控制台`

### 页面副标题

`先看整本书该推进什么，再开始写这一章。`

### 模块顺序

1. 今日推进区
   - `当前推进到：第 X 章`
   - `下一章建议任务：______`
2. 提醒区
   - `角色回归提醒`
   - `冷线提醒`
   - `伏笔回收提醒`
3. 情节线热度区
   - 主线 A
   - 支线 B
   - 隐线 C
4. 最近写回摘要区
   - `上一章关键变化`
   - `本轮新增债务`
5. 项目列表区
   - 项目卡片 1
   - 项目卡片 2

## 3. 项目真相区线框模板

`项目真相区`

## 4. 章节控制页线框模板

### 页面标题

`第 X 章 / 章节工作台`

### 页面结构

1. 主编辑区
   - 正文标题
   - 富文本编辑区
2. 控制侧栏
   - 本章控制卡
   - 相关人物切片
   - 相关主线切片
   - 伏笔切片
3. 页面底部或顶部状态区
   - 保存状态
   - 生成状态
   - 完成章节状态

### 关键按钮

- `保存正文`
- `生成控制卡`
- `生成正文初稿`
- `完成本章`

## 5. 会员页线框模板

### 页面标题

`会员与创作能力`

### 页面说明

`升级的不是更多按钮，而是更强的长篇控制能力。`

### 模块顺序

1. 免费版
   - 基础立项
   - 少量生成额度
2. 专业作者版
   - 完整控制文件
   - 控制卡无限生成
   - 连续性检查
3. 马拉松版
   - 疯狂写作模式
   - 多项目并行
   - 更强上下文与批量写回
4. CTA 区
   - `立即开通`
   - 支付二维码占位
