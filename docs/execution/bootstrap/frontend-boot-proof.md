# Quest 1.2 前端启动验收模板

## 目标

证明 `apps/web` 已经能被考官通过浏览器直接验证。

## 学生需要提交的证据

### 终端证据

- 启动命令截图
- 成功监听端口截图

### 浏览器证据

- 打开 `http://localhost:3000` 的页面截图
- 页面上必须出现明确文本锚点，例如：
  - `文枢AI`
  - `进入文枢AI`
  - `创作控制台`

## PM 黑盒验收动作

- [ ] 运行前端启动命令
- [ ] 浏览器打开 `http://localhost:3000`
- [ ] 页面能正常显示，不是浏览器错误页

## 常见错误

- 页面起了但没有任何文本锚点
- 端口不统一
- 页面是默认模板，没有产品识别信息
